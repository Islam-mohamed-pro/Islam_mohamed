## 💫 عني (About Me)

أنا **إسلام**، مطور تطبيقات هاتف متخصص في بيئة **Flutter** و لغة **Dart**. شغوف بتحويل الأفكار الإبداعية إلى تطبيقات واقعية، سلسة، وذات واجهات مستخدم جذابة (UI/UX) تعمل بكفاءة على مختلف المنصات (Android & iOS).

* 🔭 أنا أعمل حالياً على تطوير مشاريع شخصية وتجارية باستخدام **Flutter**.
* 🌱 أقوم حالياً بتعميق معرفتي في: **[اذكر شيئاً تتعلمه الآن، مثال: Bloc State Management أو Firebase Advanced]**.
* 💬 اسألني عن أي شيء يخص **تطوير تطبيقات الهاتف، Dart، أو تصميم الواجهات**.
* ⚡ حقيقة ممتعة: أنا لست مجرد مبرمج، أنا محترق لحل المشاكل (Problem Solver) وبناء تجارب مستخدم مذهلة.
